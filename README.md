
# Выжить обязан

Игра-квест с элементами визуальной новеллы, созданная студентами Вологодского колледжа связи и информационных технологий.

## Описание
Вас ждёт захватывающее приключение внутри таинственного учебного заведения, наполненного мрачными тайнами и магическими существами. Ваша задача — пройти через лабиринты учебных кабинетов, разгадать головоломки и выйти победителем!

## Цель игры
Главная цель — выжить и покинуть здание колледжа до полуночи, иначе столкнуться лицом к лицу с ужасной сущностью, которой никто ещё не видел...

## Особенности
- Атмосферная графика: Погрузитесь в мрачный мир старинного колледжа.
- Логические задания: Решайте увлекательные головоломки и интерактивные задачи.
- Необычные персонажи: Встречайтесь с разными героями, каждый из которых скрывает свою уникальную роль.
- Забавные моменты: Наслаждайтесь лёгкой долей юмора, разбавляющего напряжение.

## Минимальные системные требования
- Операционная система: Windows 10 и выше, macOS 10.15 и выше, Linux Ubuntu 20.04 LTS и выше
- Процессор: Intel Core i3 / AMD Ryzen 3 или аналогичный
- Оперативная память: минимум 2 ГБ RAM
- Место на жёстком диске: около 100 МБ свободного пространства
- Интерпретатор Python версии 3.8 и выше

## Инструкция по запуску
1. Скачайте архив с игрой.
2. Распакуйте его в удобное место.
3. Запустите исполняемый файл Game.exe.
4. Если нет .exe запускать файл Game.py
